In order to process your CruiseControl server's build reports, you must provide Build Watch with the URL of your server dashboard's RSS feed.

Note: RSS URLs are case-sensitive.

CruiseControl.rb

You can find the URL of your dashboard's RSS feed by clicking on the RSS icon in your browser's location bar. The URL should end in "projects.rss". For example:

http://builder.com/projects.rss

CruiseControl.NET

The URL of your dashboard's RSS feed is not visibly displayed through the web interface. However, you can create it by replacing "ViewFarmReport.aspx" in your dashboard's URL with "XmlServerReport.aspx".

For example, if your dashboard's URL is:

http://builder.com/ViewFarmReport.aspx

then the RSS feed's URL will be:

http://builder.com/XmlServerReport.aspx

You can confirm this by typing the URL into your web browser.

CruiseControl for Java

CruiseControl for Java provides a number of RSS feeds. Build Watch needs to be supplied with the feed that's compatible with CCTray. The link for this feed is listed on the right of the CruiseControl dashboard, in the "Tools" section. The URL will look like:

http://builder.com/dashboard/cctray.xml
